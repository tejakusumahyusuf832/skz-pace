skz_snippets

Static metadata and general details about each video. Scraped continuously but rarely changes post-publish.

video_id String (PK)
The unique 11-character identifier assigned to the video by YouTube. Acts as the Primary Key connecting all tables.
"zMJhEXR7meQ"
published_at Timestamp (TZ)
The exact date and time the video was published on the channel, including the timezone offset. Essential for calculating video age.
"2025-09-24 17:59:59+07"
video_format String
The format classification of the video natively determined during scraping.
Possible Values: "Long-form", "Short", "Live/VOD"
title String
The raw title of the video. Contains multilingual text (English, Korean) and often denotes series names like "SKZ CODE".
"Stray Kids "CREED" Video MAKING FILM"
description String
The text written in the description box of the video. Often contains credits, links, and copyright info.
tags String (CSV)
A comma-separated list of tags assigned to the video by the uploaders. Critical for NLP categorization.
"Stray Kids, bangchan challenge, changbin tiktok"

skz_stats

Time-series performance metrics. Contains multiple records per video_id, updated daily.

video_id String (FK)
Foreign key linking back to skz_snippets.
scraped_at Timestamp (TZ)
The exact timestamp when the metric snapshot was taken. Forms a composite primary key alongside video_id.
"2026-04-27 04:55:27.741242+07"
view_count Integer
Total cumulative views the video had at the moment of scraping. Stored as a string in raw JSON but must be cast to Integer.
575036
like_count Integer
Total cumulative likes the video had at the moment of scraping.
comment_count Integer
Total cumulative comments the video had at the moment of scraping.

skz_top_comments

Granular fan sentiment data. Captures the top 50 comments per video, updated daily.

comment_id String (PK)
Unique identifier for the comment assigned by YouTube.
"UgxTVDp93iucwW-UD3x4AaABAg"
text String (Multilingual)
The actual text of the comment. Contains raw emojis, line breaks, and mixed languages. Requires NLP processing.
"“i think i was born for flowers” jeongin you ARE the flower 🌼 🤍"
like_count Integer
Number of likes the specific comment has received from other users. Useful for weighting sentiment scores.
1084

skz_transcripts

Full spoken dialogue and auto-generated captions for deep textual analysis.

transcript String (Large Text)
The complete string of captions pulled from the video. Used for keyword extraction (TF-IDF) to correlate spoken topics with engagement spikes.

Engineered Features ✦

Derived metrics calculated via Python/Pandas to solve the "Time-Series & Publication Date" problem and enable advanced visualization in Power BI.

video_age_days Calculated: Float
The exact age of the video at the time of the latest data pull. Used to create cohorts (New Release vs. Catalog).
Calculation: ( MAX(stats.scraped_at) - snippets.published_at ).days
content_pillar Calculated: String
Custom categorization mapping derived via Regex from the `title` and `tags` columns. Fills the gap left by YouTube's generic 'category_id'.
Categories: 'Music Video', 'SKZ CODE', 'Vlog', 'Teaser', 'Dance Practice', 'Other'
Logic: IF title CONTAINS "SKZ CODE" THEN "SKZ CODE", ELSE IF...
lifetime_engagement_ratio Calculated: Float
The overall historical quality of the video up to the present day. Normalizes views across different videos to level the playing field.
Calculation: Latest like_count / Latest view_count
marginal_ratio Calculated: Float
The "Current Momentum". Calculates the engagement rate explicitly on the NEW views generated during the 15-25 day scraping window. Crucial for identifying older videos that are trending again.
Calculation: (Latest Likes - Earliest Likes) / (Latest Views - Earliest Views)
daily_view_velocity Calculated: Float
The average number of views the video is actively gaining per day right now.
Calculation: (Latest Views - Earliest Views) / (Days between Earliest and Latest Scrape)
positive_sentiment_percentage Calculated: Float (%)
The aggregate sentiment score per video. Generated by passing the `skz_top_comments` text through the twitter-xlm-roberta-base-sentiment transformer model.
Calculation: COUNT(Positive Comments) / COUNT(Total Comments) per video_id
